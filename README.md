# **SharedIDE**

**SharedIDE** is a collaborative coding platform that allows multiple users to code together in real time. It supports code execution directly from the browser, making it useful for pair programming, coding interviews, teaching, and teamwork.

 **Live Demo:** [https://sharedideg.onrender.com](https://sharedideg.onrender.com)

---

## **Features**

* Real-time collaborative editor
* Code execution with terminal-style output
* Multiple language support
* Simple and clean UI for easy collaboration
* Browser-based, no setup required

---

## **Tech Stack**

**Frontend:**

* React
* JavaScript
* HTML / CSS

**Backend:**

* Node.js
* Express

**Real-time:**

* WebSockets
* Socket.IO

**Deployment:**

* Render



## **Installation & Setup**

# Clone the repository
git clone https://github.com/abishekak18/SharedIDE.git

cd SharedIDE

# Install server dependencies
cd server

npm install

# Install client dependencies
cd ../client

npm install


### **Start Development**


# Start backend
cd server

npm start

# Start frontend
cd ../client

npm start


## **Usage**

1. Open the live link or local development link.
2. Create or join a shared coding session.
3. Code collaboratively with friends.
4. Run the code to test execution.
